Builder of Network Enabled Video Tech

Cody is currently working on a technology stack to produce broadcast-grade video content using commodity computing and networking hardware. This tech will be used to provide long form production services for events such as conferences and expos, and charity gaming events.

Engineer of Cloud Platforms

Cody also currently working on a private/hybrid cloud environment for hosting a web application that powers clinical trials. His efforts include migrating a software stack from a reactively grown and aging infrastructure to a software defined cloud stack using technologies such as Ansible, VyOS, vSphere, Ubuntu and Red Hat Enterprise Linux.

Weaver of Beats
